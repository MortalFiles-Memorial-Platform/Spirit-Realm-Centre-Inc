# Spirit Realm Centre Web App

You can access the web version of the Spirit Realm Centre Management System directly from your browser:

**Direct Link:**
https://src-web-demo.netlify.app/

> If you want your own custom URL, deploy the app to Netlify, Vercel, or GitHub Pages. See below for instructions.

---

## How to Deploy Your Own Web App

### 1. Netlify
- Go to https://netlify.com
- Click "Add new site" > "Import from GitHub"
- Select your repository and follow the prompts
- Your app will be live at a custom Netlify URL

### 2. Vercel
- Go to https://vercel.com
- Click "New Project" > "Import GitHub Repo"
- Select your repository and follow the prompts
- Your app will be live at a custom Vercel URL

### 3. GitHub Pages
- Go to your repository settings
- Enable GitHub Pages for the main branch
- Your app will be live at https://yourusername.github.io/yourrepo/
